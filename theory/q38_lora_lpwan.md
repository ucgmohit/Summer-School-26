# Q38: LoRa & LPWAN – Long Range IoT

LPWAN (Low Power Wide Area Network) is a communication technology  
designed for long-range IoT communication with very low power consumption.

It is widely used in smart agriculture, smart cities, and remote monitoring systems.

LoRa (Long Range) is a popular LPWAN technology that enables communication  
over distances of 10–15 km in rural areas while consuming very low power.

It works using spread spectrum modulation, which spreads signals across a wide frequency band,  
making communication more resistant to noise and interference.

A key concept in LoRa is the Spreading Factor (SF).  
Higher SF increases communication range but reduces data rate.  
Lower SF increases speed but reduces range.

This trade-off allows optimization based on application needs.

### Comparison:
- LoRa: Long range, very low power, low data rate  
- Zigbee: Short range (~100m), mesh network  
- Z-Wave: Home automation, low interference  

### Real-world Example in India:
LoRa is used in smart agriculture systems in states like Punjab and Maharashtra.  
It helps in soil monitoring, irrigation control, and weather tracking using remote sensors.

These sensors send data to a central gateway, which then forwards it to cloud servers.

## Conclusion
LoRa is ideal for large-scale IoT deployments where devices are far apart  
and battery life is a critical factor.
