# Bluetooth LED Controller

## Hardware
- Arduino UNO
- HC-05 Bluetooth Module
- Red LED
- Green LED
- Blue LED
- 3 × 220Ω Resistors

## Pairing Instructions

1. Power ON the HC-05.
2. Open Bluetooth settings on your Android phone.
3. Search for **HC-05**.
4. Pair using PIN:
   - 1234
   - or 0000
5. Open **Serial Bluetooth Terminal**.
6. Connect to HC-05.
7. Send the following commands:

- R → Red LED ON
- G → Green LED ON
- B → Blue LED ON
- X → All LEDs OFF
- F → Flash all LEDs 5 times
